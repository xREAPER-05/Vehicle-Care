# 🚗🏍️ Vehicle Care

Aplicación móvil desarrollada con **Ionic** y **Angular** como proyecto de formación del programa **Análisis y Desarrollo de Software (SENA)**.

Vehicle Care tiene como objetivo ayudar a propietarios de **motocicletas y automóviles** a gestionar la información de sus vehículos, realizar el seguimiento de recorridos mediante geolocalización y llevar un control del mantenimiento preventivo desde una sola aplicación.

---

# 📌 Estado del proyecto

🚧 **En desarrollo**

Actualmente el proyecto se encuentra en la fase de implementación y pruebas de la funcionalidad de geolocalización para calcular la distancia recorrida durante un trayecto.

---

# 🎯 Objetivo

Desarrollar una aplicación móvil que permita administrar la información de motocicletas y automóviles, registrar recorridos mediante geolocalización y facilitar el control del mantenimiento preventivo del vehículo.

---

# ✨ Funcionalidades previstas para la versión 1.0

- 🚗 Registro de automóviles.
- 🏍️ Registro de motocicletas.
- 📍 Registro de recorridos mediante geolocalización.
- 📏 Cálculo automático de kilómetros recorridos.
- 🔧 Gestión de mantenimientos preventivos.
- 💾 Almacenamiento de la información del vehículo.

---

# ✅ Avance actual

- Configuración inicial del proyecto con Ionic y Angular.
- Desarrollo de la estructura base de la aplicación.
- Implementación de pruebas de geolocalización.
